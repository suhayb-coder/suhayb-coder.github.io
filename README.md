# Suhayb Said Swaleh - Personal Portfolio 🚀

Welcome to my personal developer portfolio! This repository contains the source code for my portfolio website, which highlights my skills, academic journey, and contact information.

## 🌟 Overview

I am a passionate computer science student and software developer based in Zanzibar. My focus is on learning and practicing modern software development, building practical projects, and growing as a software engineer.

You can view the live site here: [https://suhayb-coder.github.io/](https://suhayb-coder.github.io/) *(Make sure to update this link if your URL changes!)*

## 🛠️ Built With

- **HTML5**: Semantic markup for the page structure.
- **Tailwind CSS**: Used for rapid, responsive, and modern styling.
- **Vanilla JavaScript**: For simple interactive elements (like the scrolling navigation highlight).

## ✨ Features

- **Modern Dark Mode Aesthetic**: Featuring a deep space-themed background with glassmorphism UI elements.
- **Responsive Layout**: Designed to look great on desktop, tablet, and mobile devices.
- **Unified Action Buttons**: Vibrant, animated gradient buttons with hover and active states.
- **Academic Timeline**: Highlights my educational milestones.
- **Downloadable CV**: Easy access for recruiters to download my resume.

## 💻 Local Development

If you'd like to run this site locally:

1. Clone this repository:
   ```bash
   git clone https://github.com/suhayb-coder/Suhayb-Portfolio.git
   ```
2. Navigate into the directory:
   ```bash
   cd Suhayb-Portfolio
   ```
3. Open `index.html` in your favorite web browser, or use a local development server like Live Server (for VS Code) to serve the file.
